# Mily 파일럿 테스트 안내

## 앱 소개

Mily(밀리)는 가족과 개인이 돈에 대해 **'덜 싸우고, 더 계획하고, 함께 돌아보는'** 습관을 만드는 주 1회 5분 소비 회고 앱입니다.

> 슬로건: "미루지 않는 경제 대화, Mily"

**중요:** Mily는 금융 서비스가 아닙니다. 실제 돈의 이동은 없으며, 소비 습관 코칭에 집중합니다.

---

## 설치 방법

### iOS (TestFlight)
1. iPhone에서 **TestFlight** 앱을 설치하세요 (App Store)
2. 아래 초대 링크를 클릭하세요
   - 초대 링크: `[TestFlight 초대 링크 — 배포 후 업데이트]`
3. TestFlight에서 **Mily** 앱을 설치하세요
4. 앱을 실행하고 카카오 또는 Apple 계정으로 로그인하세요

### Android (내부 테스트)
1. 아래 링크에서 APK 파일을 다운로드하세요
   - 다운로드 링크: `[APK 다운로드 — 배포 후 업데이트]`
2. 설치 후 앱을 실행하세요

## 테스트 시나리오

### 필수 과제

| 주차 | 과제 | 예상 소요 |
|------|------|-----------|
| 1주차 | 온보딩 완료 + 첫 체크인 3회 | 15분 |
| 1주차 | 가족 연결 (자녀 또는 부모와) | 5분 |
| 2주차 | 첫 주간 회고 완료 | 5분 |
| 2주차 | 요청 카드 보내기 1회 | 3분 |
| 3주차 | 칭찬 카드 보내기 1회 | 2분 |
| 4주차 | 월간 소비 돌아보기 | 5분 |

### 선택 과제

- 꿈 설계소에서 목표 시뮬레이션
- 세뱃돈/명절돈 기록 (해당 시즌)
- 가족 약속 기록함에서 새 약속 만들기
