# Mily Android 설치 방법

## APK 직접 설치 (카카오톡 전송)

### 1단계: 파일 다운로드
카카오톡으로 받은 **Mily.apk** 파일을 탭하세요.

### 2단계: "출처를 알 수 없는 앱" 허용
설치 시 보안 경고가 뜨면 아래 순서로 허용해주세요:

**Android 13 이상:**
1. 설정 → 앱 → 특별한 앱 권한 → 알 수 없는 앱 설치
2. 카카오톡 (또는 다운로드 앱) → **허용** 토글

**Android 12 이하:**
1. 설정 → 보안 → 알 수 없는 출처 → **허용**

### 3단계: 설치 완료
1. 설치가 완료되면 **열기** 버튼 탭
2. 홈 화면에 **Mily** 아이콘 생성 확인

### 4단계: 시작하기
1. Mily 앱 실행
2. **카카오 로그인** 또는 **이메일 로그인**으로 시작
3. 온보딩 안내를 따라 설정 완료

## 자주 묻는 질문

### "이 앱은 유해할 수 있습니다" 경고가 떠요
- Play Protect 경고입니다. 정상이에요.
- **무시하고 설치** → 확인을 눌러주세요.
- TestFlight/내부 테스트 버전이라 발생하는 정상 경고입니다.

### 설치가 안 돼요
- 저장 공간이 충분한지 확인해주세요 (최소 150MB)
- Android 7.0 (API 24) 이상이 필요합니다
- 설정 → 앱 → 카카오톡 → 권한 → 저장공간 허용 확인

### 업데이트는 어떻게 하나요?
- 새 버전 APK를 카카오톡으로 다시 받아 설치하면 됩니다
- 기존 데이터는 유지됩니다
