# 🌱 Mily — 미루지 않는 경제 대화

> **돈 얘기, 이제 싸움 말고 카드로.**

Mily는 가족 간의 소비 습관을 함께 만들어가는 AI 코칭 앱입니다.  
지출을 통제하는 것이 아니라, 대화하고 이해하는 방식으로 접근합니다.

---

## 📱 주요 기능

### 🗂 일일 체크인 (Daily Check-in)
- **Quick / Standard / Detailed** 3가지 모드 지원
- 오늘의 소비를 간단하게 기록하고 돌아보기

### 📋 요청 카드 (Request Card)
- 자녀 → 부모 용돈 협상 카드 발송
- 감정이 아닌 카드로 대화하는 경제 교육

### 🔄 고정 지출 자동 확인 (Recurring Expense)
- 반자동 확인 플로우로 정기 지출 누락 방지
- 월 결산 자동 정리

### 📊 역할별 리포트 (Report System)
- 3가지 역할(개인 / 자녀 / 부모) × 2가지 주기(주간 / 월간) 리포트
- 가족 소비 패턴을 한눈에 파악

### 🔒 프라이버시 설정 (Privacy)
- 3단계 공유 모드 (비공개 / 요약 공개 / 전체 공개)
- 7일 만료 옵션으로 민감 정보 자동 정리

### 🎯 약속 루프 (Promise Loop)
- 소비 약속 이행 추적
- 잘 지킨 순간 기록 + 부모 알림

### 🧠 스마트 예산 (Smart Budget)
- AI 기반 지출 패턴 분석
- 다음 달 예산 자동 제안

### 🔔 FCM 푸시 알림
- 체크인 리마인더, 요청 카드 도착, 약속 만료 알림

### 📡 기기 동기화 (Device Sync)
- 가족 구성원 간 실시간 데이터 동기화

### 🎓 졸업 시스템 (Graduation)
- 소비 습관 목표 달성 시 단계별 졸업 인증
- 장기 습관 형성 동기 부여

### 📈 모니터링 (PostHog)
- 사용자 행동 분석 및 제품 개선 데이터 수집

## 🧬 DNA 원칙

Mily는 다음 원칙을 절대 위반하지 않습니다.

- ❌ 금융 실행 없음 (송금, 결제, 계좌 연동 없음)
- ❌ 감시 없음 (실시간 위치 추적, 강제 열람 없음)
- ❌ 데이터 판매 없음
- ✅ Behavior-First Coaching — 행동 변화가 목적, 통제가 목적이 아님

## 🗺 서비스 비전

```
Stage 1 — Tool      가족 소비 회고 도구 (현재)
Stage 2 — Habit     반복과 루틴으로 습관 형성
Stage 3 — Community "밀리 세대" — 경제 대화가 자연스러운 가족 문화
```

> Stage 3 기능은 Stage 1 PMF 달성 전까지 개발 금지
